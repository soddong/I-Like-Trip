# ✈️ Enjoy Trip (조아요행)

## 프로젝트 소개

- 국내 여행자를 위해 지역별, 키워드별로 관광지 정보를 제공하는 서비스 입니다.
- 관광지 정보를 지도 기반으로 제공하여 사용자가 직관적으로 사용할 수 있습니다.

## 팀원 소개

- 서종원
- 최소현

## 요구사항
Front-End 부분을 Vue로 변경하여 구현
